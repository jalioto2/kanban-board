# Kanban Task Manager

A React + Supabase Kanban board with drag-and-drop functionality.

## 🛠️ Quick Setup
1. Clone the repo: `git clone [repo-link]`
2. Go to folder: `cd kanban-board`
3. Install: `npm install`
4. Add your `.env` file with `VITE_SUPABASE_URL` and `VITE_SUPABASE_ANON_KEY`.
5. Run: `npm run dev`
