# Kanban Task Manager

A React + Supabase Kanban board with drag-and-drop functionality.

## 🛠️ Quick Setup
1. Clone the repo: `git clone https://github.com/jalioto2/kanban-board`
2. Go to folder: `cd kanban-board`
3. Install: `npm install`
4. Add your `.env` file with `https://xhizkjkxqaonfsjrkcjm.supabase.co` and `sb_publishable_8e46z3Ryw7kTc-zs9q-vng_QHAItDDE`.
5. Run: `npm run dev`
